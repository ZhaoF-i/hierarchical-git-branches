# Changelog

## 1.0.0

- Add branch filtering.
- Auto-reveal the current branch in the tree.
- Show local tracking state, ahead/behind counts, and last commit time.
- Add checkout command copying, current branch push, selected branch update, branch rename, and branch pinning.
- Require typed confirmation when deleting a remote branch.

## 0.0.5

- Show the current branch as a fixed top item in the branch tree.
- Show the current branch in the VS Code status bar.
- Show detached HEAD state with a short commit SHA.

## 0.0.4

- Create a new branch from the right-clicked branch instead of the current branch.
- Add a guarded delete action that removes matching local and remote branches.

## 0.0.2

- Create a new branch from the right-clicked branch instead of the current branch.
- Add a guarded delete action that removes matching local and remote branches.

## 0.0.1

- Add hierarchical local and remote Git branch views.
- Add branch checkout from the tree.
- Add branch creation from current branch.
- Add fetch all, refresh, and copy branch name commands.
- Support Remote SSH by running as a workspace extension.
